order of sytling in CSS is "deepest out", in other words the rules applies closest to that element. element specific styling will be applied on top of embedded styling (from html head) onto styling from external file, onto browers defaut styling. for web developpers conflict resolution would be defining the second element specifically to override the conflict. the last factor is the !important tag which overrides any more specific rules. Other methods are from the addition of a class to a specific element of your own type. lastly there are id selectors to assigns styling to a certain element.
